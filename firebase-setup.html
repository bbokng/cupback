🌐 CupBack - 모든 기기에서 데이터 공유하기

⚠️ 현재 상황: 각 기기마다 데이터가 따로 저장되고 있어요.

🚀 Firebase로 해결하기 (추천)

1단계: Firebase 프로젝트 생성

  1. Firebase Console에 접속
  2. "프로젝트 추가" 클릭
  3. 프로젝트 이름: "cupback-app" 입력
  4. Google Analytics 비활성화 (선택사항)
  5. "프로젝트 만들기" 클릭

2단계: 웹 앱 등록

  1. Firebase Console에서 "웹 앱 추가" 클릭
  2. 앱 닉네임: "cupback-web" 입력
  3. "앱 등록" 클릭
  4. 설정 코드 복사 (나중에 사용)

3단계: Firestore 데이터베이스 생성

🎯 이 단계가 가장 중요합니다!
  1. 왼쪽 메뉴에서 "Firestore Database" 클릭
    📍 Firebase Console 왼쪽 사이드바에서 "Firestore Database" 찾기
  2. "데이터베이스 만들기" 클릭
    🔥 중앙에 있는 파란색 "데이터베이스 만들기" 버튼 클릭
  3. 보안 규칙 선택
    🛡️ "테스트 모드에서 시작" 선택 (나중에 보안 설정 가능)
  4. 위치 선택
    🌏 "asia-northeast3 (서울)" 선택 (한국에서 가장 빠름)
  5. "완료" 클릭
    ✅ 데이터베이스 생성 완료까지 1-2분 대기
✅ 완료 확인:
  • Firestore Database 페이지에 "데이터" 탭이 보임
  • 왼쪽에 "컬렉션 시작" 버튼이 보임
  • 상단에 프로젝트 ID가 표시됨

4단계: 코드 수정

Firebase 설정이 완료되면 제가 코드를 수정해드릴게요!

다음 단계:
1. 설정 정보를 firebase-config.js에 입력
2. HTML 파일에 Firebase SDK 추가
3. 스크립트 파일 교체
✅ 완료 후 효과:
  • 모든 기기에서 같은 데이터 공유
  • 실시간으로 업데이트
  • 무료로 시작 가능 (월 1GB까지 무료)
  • 안전한 데이터 저장
